## Git流程

1. 你先通过 git clone https://github.com/LIJALEN23/Travel-System.git 拿到仓库(你看到这一条了，说明你已经执行过了)

2. 现在你本地有项目了，然后执行 git checkout -b youname-feature 。这样是为了避免大家都在main分支上开发，会有很多冲突

3. 你每次完成了一个阶段的开发，就需要提交，这样大家才能获取的你进度。提交就是 git add..git commit....，最重要的是git push origin yourname-feature(不要提交到主分支即main)。

4. 每次你需要与其他人对齐进度时，在main分支执行 git pull 即可拉取到main分支的最新进度(我们会把所有你yourname-feature分支上的内容合并到main分支中)
